Streamline Your RabbitMQ Message Handling with Bunnet

The open-source client that lets you connect, visualize, and manage AMQP messages with precision.

Bunnet is a Work in Progress

Bunnet is currently in beta, and some bugs are to be expected. Please help us improve by reporting any issues or suggesting new features on our GitHub issues page.

Why Choose Bunnet?

Visualize and Filter Messages

Gain clear insights into your message streams with powerful visualization tools and customizable filters.

Seamless Integration

Integrates smoothly into your existing setup with auto-deleting queues to keep your system pristine.

Interactive Messaging

Interact directly with your messages: send updates back to exchanges and manipulate JSON data effortlessly.
